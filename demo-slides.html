Umělá inteligence, kterou máte doma

Program, který si umí povídat, překládat a shrnovat text — a to celé na tomhle počítači. Bez internetu, bez cloudu, bez poplatků.

PHP aplikace + malý jazykový model + Docker · ~15–20 min
01 · Základ

Co je to vlastně „AI"?

Velmi sečtělý pomocník, který přečetl obrovské množství textů a naučil se z nich, jak na sebe slova navazují. Jako napovídání slov v mobilu — jen mnohonásobně chytřejší.

a

Není to kouzlo

Je to statistika a hodně přečteného textu.

b

Nemá vlastní vůli

Jen odpovídá na to, co dostane. Nemyslí, necítí.

c

Občas se splete

Někdy si něco „vymyslí". Je to vlastnost, ne chyba.

04b · Živě — pro techničtější publikum

Příklad 4 · Nálada textu jako data

Výstup nemusí být věta pro člověka. Model umí vrátit strukturovaný JSON — přesně to, co potřebuje další program.

Vstup
Ta nová kavárna je naprosto skvělá, obsluha byla milá a káva výborná!
Výstup (JSON)
{ "sentiment": "pozitivní", "skore": 0.95, "duvod": "chvála podniku, obsluhy i kávy" }

→ Tohle už si přečte jiná aplikace, ne jen člověk. Základ automatizace.

04d · Živě — pro techničtější publikum

Příklad 7 · Z věty rovnou PHP funkce

Napíšete, co má funkce dělat — model vygeneruje hotový kód i s komentářem. Pomocník pro programátora, ne náhrada.

Vstup (popis)
sečte dvě čísla
Výstup (PHP)
/** Sečte dvě čísla. */ function secti(float $a, float $b): float { return $a + $b; }

→ CLI varianta: php src/cli.php 7 "sečte dvě čísla"

07 · Časté otázky publika

Na co se lidé ptají

Špehuje mě to? Posílá něco ven?
Ne. Model běží lokálně — offline ukázka to dokazuje.
Je to zadarmo?
Ano. Model i software jsou open-source, žádné poplatky.
Je to stejné jako ChatGPT?
Princip ano, ale ChatGPT běží na cizích serverech a je mnohem větší.
Nahradí to lidi?
Je to nástroj — pomůže s rutinou, výstup je ale třeba kontrolovat.
Proč je to někdy pomalé?
Použili jsme záměrně malý model, aby běžel i na slabém počítači.
Může se to splést?
Ano, občas si „vymyslí" — proto pomocník, ne neomylný zdroj pravdy.
Závěr

AI nemusí být tajemná služba někde na internetu.

Může to být obyčejný program, který si spustíte doma a plně mu rozumíte — u vás, bez internetu, zadarmo a soukromě.
